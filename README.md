** Fonctionnalités

Design Interactif : Les cartes pivotent à 180° au survol de la souris (hover effect).

Effet 3D Réaliste : Utilisation des propriétés perspective et transform-style: preserve-3d pour donner de la profondeur.

Contenu Riche : Présentation de sites historiques tels que la Casbah d'Alger, le Jardin d'Essai du Hamma, et la Basilique Notre-Dame d'Afrique.

Layout Responsive : Utilisation de Flexbox pour un alignement fluide des cartes sur tous les écrans.

** Technologies utilisées
 
HTML5 : Structure sémantique du contenu.

CSS3 :

Animations et transitions fluides.

Gradients de couleurs modernes.

Transformations 3D (rotateY, translateZ).

Polices personnalisées via Google Fonts (Montserrat & Poppins).

** Aperçu du rendu

Le projet affiche des cartes de 320x320px.

Face avant : Une image immersive du lieu.

Face arrière : Un panneau coloré avec un dégradé (rose/jaune) contenant le titre et une description détaillée du monument.
